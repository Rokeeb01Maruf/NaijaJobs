Project Name: NaijaJobs
Project description: A capstone project that c
is about job listing and applying for job which features rolw base authorization with first role "Job seeker" and the other role "Employer".
In which job seekers are the user that got on the site to find a job, access job list and details, apply for it and wait for application status whether accepted or rejected while Employer create employer profile, create a job, view the lists of applicants and either accept or reject applicants.
Project stack: Django templates for frontend, Django for backend and Django SQLite for the database, Javascript DOM for validation
